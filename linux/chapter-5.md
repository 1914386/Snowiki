# Chapter 5

`su -c '명령어'`: root 권한으로 명령어 수행

* 노틸러스(`nautilus`): 그놈 데스크톱 환경에서 제공되는 파일 관리자
* 에볼루션(`evolution`): 이메일 클라이언트
* 피진(`pidgin`): 메신저 서비스
* `gftp`: FTP 클라이언트
* `gnome-control-center`: 사운드 설정
* `rhythmbox`: 음악 재생
* `totem`: 동영상 플레이어
* `gedit`: 메모장
* 다중 문서 보기: `evince 파일명`
* 브라세로(`brasero`): CD/DVD 레코딩
* `gimp`: 그래픽 편집 (윈도우의 포토샵)
* `eog`: 그래픽 뷰어
* LibreOffice
  * Writer = 워드
  * Calc = 엑셀
  * Impress = 파워포인트
  * Base = 액세스
  * Draw: 그리기 툴
  * Math: 수식 작성 툴
